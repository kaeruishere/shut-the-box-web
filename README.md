Klasik zar oyunu **Shut The Box**'ın web tarayıcıları üzerinden oynanabilen, Firebase Hosting entegrasyonuna sahip responsive web sürümüdür.

## 🌐 Özellikler

- **Kurulumsuz Oynanış:** Tarayıcı üzerinden anında erişim.
- **Responsive Arayüz:** Masaüstü, tablet ve mobil cihazlarla tam uyumlu CSS tasarımı.
- **Firebase Dağıtımı:** Hızlı ve güvenli Firebase Hosting entegrasyonu (`firebase.json`).

## 🚀 Dağıtım (Deploy)

Projeyi Firebase Hosting üzerinde yayınlamak için:
```bash
npm install -g firebase-tools
firebase login
firebase deploy
```
